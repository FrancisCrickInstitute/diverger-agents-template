Title: Label-free cell cycle stage classification using phase imaging with machine learning




Abstract: Traditional cell cycle stage classification relies on fluorescence microscopy, utilising biosensors like PCNA or FUCCI for dynamic assessment [1][2]. Unfortunately, phototoxicity and photobleaching limit the efficacy of fluorescence-based approaches during extended live cell time-lapse experiments. We present a label-free cell cycle determination workflow using inexpensive, open-source quantitative phase imaging (QPI), implemented with polarisation differential contrast microscopy (pDPC) [3], combined with machine learning. 
For this, we acquired timelapse fluorescence microscopy data of PCNA-mScarlet expressing cells, and also acquired QPI images at each timepoint. The fluorescence data served as a ground truth for cell cycle stage, facilitating training of a ResNet-50 architecture for identification of cell-cycle specific features solely from QPI images. This required co-registration of the two images, coupled with segmentation (CellPose) and tracking (TrackMate) of individual cells over time. Automatic fluorescence image feature extraction was undertaken using custom Python scripts to produce an annotated ground truth dataset. 
This approach demonstrates high accuracy (~90%) when classifying live cells into G1, S, G2 and M stages using only QPI images. pDPC can enable QPI on almost any fluorescence microscope, widely enabling studies of single-cell dynamics throughout the cell cycle, e.g., to probe heterogeneity in image-based high-throughput drug screening.
