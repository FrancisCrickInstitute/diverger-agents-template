Title: A workflow for automated cell-tracking

Abstract:
Analysis of cell motility require the ability to reliably track cell movement for the duration of the experiment. Automated cell tracking has become more feasible with the development of new open-source deep-learning tools designed specifically for cell segmentation [1]. These tools can automatically detect cells or objects in biomedical images and provide a segmentation that can be used by particle tracking packages to determine cell trajectories throughout a live-cell experiment [2].  

Here we present an experimental protocol that allowed us to implement a fully automated pipeline for tracking glioblastoma cells during a live-cell experiment. We used a fluorescent dye to stain the cell nuclei, allowing us to image the cells using a fluorescent microscope. The analysis sequence uses a retrained cellpose model to segment the cell nuclei [1], and the Btrack package in python to track the cell masks generated [3]. The script is completely automated and will export an experimental report from the inputted data, alongside the cell segmentation and tracking information for each time-lapse image analysed. To make the analysis sequence as user-friendly as possible, the python script is contained within a jupyter notebook [4].
